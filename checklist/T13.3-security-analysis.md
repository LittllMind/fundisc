### 🔴 Corrections nécessaires

#### 1. Routes des fonds (lignes ~137-155)
**Problème** : Tests utilisent `{id}` mais route définit `{fond}` (binding modèle).

**Correction** : Soit changer les tests pour passer un objet, soit utiliser l'ID directement mais la route attend le binding.

```php
// Actuel dans les tests:
"/fonds/{$fond->id}/stock"

// Route dans web.php:
"/fonds/{fond}/stock"  // binding modèle

// Solution: La route fonctionne avec ID, mais le test vérifie 403
// Vérifier que la route retourne bien 403 pour les non-admin
```

#### 2. Security Headers (lignes ~207-225)
**Problème** : Les headers X-Frame-Options, X-Content-Type-Options, CSP ne sont pas configurés.

**Solution** : Installer bepsvpt/secure-headers ou configurer manuellement dans un middleware.

**Déjà corrigé** : Le test vérifie maintenant token invalide = 419.
