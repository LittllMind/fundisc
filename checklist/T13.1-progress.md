## ⚠️ Problème Test DB

Les tests échouent avec erreurs:
- `SQLSTATE[42S01]: Base table 'users' already exists`
- Conflits entre tests en parallèle

**Cause**: Configuration phpunit.xml avec `RefreshDatabase` en mode parallèle

**Solution nécessaire**: 
Configurer phpunit.xml pour tests séquentiels:
```xml
<env name="PARALLEL_TESTING" value="false"/>
```

Ou utiliser SQLite in-memory qui est plus fiable pour tests parallèles
